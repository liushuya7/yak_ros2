\doxysubsection{Detailed Description}
This class visualizes useful information for yak\+\_\+ros1\+\_\+node. It publishes the tsdf volume as a bounding box as well as the resultant mesh. 

Note that these markers are published every second by a R\+OS timer that is started when the feature is set and stopped when the feature is cleared.
